# Imperial-Scribe

## How GPT-5.6 was used

GPT-5.6 powers the intelligent productivity assistant.

It is used to:

- Generate daily reflections
- Break down large tasks into smaller actionable subtasks
- Summarize daily achievements
- Provide motivational coaching
- Generate historical "royal chronicle" entries
